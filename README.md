# LightBender
A LightBender is a form of FLS with an actuated rod as its lighting primitive.
A swarm of LightBenders illuminate graphics and animations visible to the naked eye.
When equipped with a cage, a LightBender facilitates safe, bare-hand interaction. 

Authors: Hamed Alimohammadzadeh(halimoha@usc.edu), Shuqin Zhu (shuqinzh@usc.edu), and Shahram Ghandeharizadeh (shahram@usc.edu)

## Features

* Illuminations using a swarm of LightBenders, see below.
* Authoring tool in the form of a Blender add-on, see below and the instructions in the local [authoring](authoring/) directory.
* Bare-hand interaction with LightBenders, see below and the instructions in the local [Interaction](Interaction/) directory.
* Upload the video recording and log files generated by an illumination to a Google drive for further analysis, see [orchestrator](orchestrator/) directory.
